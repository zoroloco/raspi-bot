#Requirements:

- Raspberry Pi Zero v1.3 (camera version)
- Raspberry Pi Zero camera cable
- Raspberry Pi Camera module
- Lots of actobotics!
- Maestro 12 servo USB controller (Pololu)
- 5 volt 10amp power supply

#To run:
systemctl start|restart|stop raspy.service
